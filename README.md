# MISSING ENTRY [LD45]

![Missing entry](https://i.imgur.com/sUQiNXQ.png)

Created in 48h for the 45th Ludum Dare

Missing entry is a story written by you and others that started with nothing and can do in any direction.

Find a [MISSING ENTRY] to add your part of the story.
